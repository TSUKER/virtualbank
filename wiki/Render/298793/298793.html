Project Description
Virtual bank is a simple web-based budgeting and finance system. It's developed in C# and ASP.NET 4.0. I created it because I wanted a solution that was geared specifically for my own needs and not too complicated.

The Virtual Bank project is a simple web application that I thought up after working with one after another complicated and confusing applications for managing my finances. It's not automatic, and it doesn't connect to any banking systems; the whole point of the project is to be simple and easy to use. Once I'm finished with the version I'm hoping that my budgeting will work out a lot better! :-)
